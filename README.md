# simple-backup

** Copy files from one place to another as a backup **

```

source:
  paths:
    - super projet 1: C:/Users/denis/wks-tools/simple-backup/env/data/folder-1
    - super projet 2: C:/Users/denis/wks-tools/simple-backup/env/data/folder-2
  exclude:
    - *.mkv
    - *.jar

target:
  path: E:/tmp
  purge-after: 180
    

```
